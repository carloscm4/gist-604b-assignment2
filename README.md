# gist-604b-assignment2
GIST 604B Assignment 2: QGIS Desktop GIS Projects

## Overview
This repository contains a collection of GIS tutorials completed in QGIS to build practical geospatial skills. The tutorials walk through common tasks like creating maps, working with different data types, and running basic spatial analysis. Along the way, I worked with both vector and raster data and learned how to manage and visualize it effectively. Overall, this project shows my progress in learning how GIS tools are used in real-world situations.

## Software Environment
- **QGIS Version Used:** QGIS Desktop 4.0.0  
- **Operating System:** Windows 11 Home  
- **Plugins Used:**  
  - none

## Skills Demonstrated
- Working with coordinate reference systems (CRS)  
- Importing and organizing spatial data  
- Styling vector and raster layers  
- Editing and managing attribute tables  
- Performing table joins and spatial joins  
- Running spatial queries  
- Basic raster and vector analysis  
- Calculating distances and line lengths  
- Creating map layouts for presentation  
- Improving map readability with symbology  

## Reflection
Working through these tutorials helped a lot of the core GIS concepts start to make sense, especially how coordinate systems work and why they matter. It was also helpful to see how different types of data can be combined and used together in one project. One of the harder parts was dealing with projections and making sure everything lined up correctly, since small mistakes could throw things off. I also found joins and spatial queries a bit tricky at first, but they got easier with practice. Overall, the hands-on work made a big difference in understanding how QGIS actually works. I can see myself using these skills in future classes or projects that involve mapping or spatial data. This experience gave me a solid starting point to build on.
