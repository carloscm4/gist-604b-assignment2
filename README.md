## Project Description  
In this assignment, I worked with spatial data using Python and GeoPandas. The main goal was to get comfortable loading geospatial datasets, cleaning them up, and running some basic spatial operations to better understand the data.

## Tools and Technologies  
- Python  
- GeoPandas  
- Pandas  
- Jupyter Notebook  

## What I Did  
- Loaded and looked through the datasets to understand what they contained  
- Cleaned up the data and fixed any formatting issues  
- Reprojected the data so everything used the same coordinate system  
- Ran a few spatial operations like filtering and joins  
- Exported the results for later use  

## How to View / Run  
- Clone the repo:  
  `git clone https://github.com/carloscm4/gist-604b-assignment2.git`

- Run the cells in order to see the full workflow
